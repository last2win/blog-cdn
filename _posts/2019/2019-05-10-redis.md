里面有关键的一句：
`redis-server[29517]: 29517:C 10 May 14:48:08.460 # Fatal error, can't open config file '/etc/redis/redi`
这句话让我想起了我修改了redis.conf，现在redis.conf是root用户，而redis用户是没有权限读取的。

解决方法：
```js
chmod 644 /etc/redis/redis.conf
service redis-server start
```
